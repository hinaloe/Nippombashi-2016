Change the font of TwentySixteen to Gothic that was adapted to the Japanese. Difficult adjustment is not required.

== Description ==

It will change the font set of TwentySixteen to those easy to read beautifully in Japanese without troublesome setting.

Font set is with reference to the following page.
[どの環境でも「きれいなフォント」をCSSのfont-familyで指定するにはどうすべきか？ - kurotei databook](http://blacktails2.hatenablog.jp/entry/2014/08/30/005150)
